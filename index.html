# Selector Seleccionado Ejemplo
1 $("*") Selecciona todos los elementos
2 $("#") Selecciona un elemento que se identifique con un id
3 $(".") Selecciona todos los elementos que tengan asignado una clase
4 $("div") Selecciona todos los elementos que sean de tipo DIV
5 $("p") Selecciona todos los elementos que sean de tipo P
6 $("input") Selecciona todos los elementos que sean de tipo input
